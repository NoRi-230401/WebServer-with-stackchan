「おきもく」さんが、まとめてくれた公開物wikiのイラストのデータをもとに
WebServer-with-Stackchanで使えるように加工したファイルを同梱しました。

https://okimoku.com/wiki/%E3%82%A4%E3%83%A9%E3%82%B9%E3%83%88


iconsフォルダ下の ( 01 to 10)フォルダ内にある icon.gifファイルを 
SPIFFSにUploadするとHome画面でイラストが表示されます。

※ 10個のファイルを同梱していますが、ファイル名を"icon.gif"にして、
SPIFFSにUploadすれば独自のイラストも表示できます。
あまり、処理に負担がかからないように、縦横400ピクセル程度、ファイル
の大きさが、50kbyte程度がいいかと思います。
